Display

There are several options for display: block, inline, and inline-block. Some items are block level by default, meaning the take up the entire width of the page by default. For example, these three boxes are made through div elements and are block level by default. Each box is on its own line.

Inline-Block

Setting the blocks to inline-block will allow them to be on the same "line" and allow them to be next to other elements. The element only takes up as much space as is needed.

Inline

Images are defaulted to inline, so nothing special needs to be done to put them on the same line.

Settings the images to display: block will put each image on its own line.
