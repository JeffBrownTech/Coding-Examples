Relative Position

The middle brown box is displayed using position: relative.

It is set to top: 50px and left: 50px.

This allows setting an element relative to its default static position on the webpage.

This moves the box down and to the right.

Absolute Position

The middle brown box is displayed using position: absolute.

This means the other elements ignore the absolute element and pretend it's not there. The brown brown will be anchored to the closest parent element. The green box is behind the brown box.

The brown box is offset to show the green box underneath it.
