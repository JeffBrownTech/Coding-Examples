Flexbox

Flexbox allows positioning elements on a page in a flexible way. Adding display: flex to an element will turn it into a flex container. This enables aligning any children of that element into rows or columns. Flexbox is used for positioning elements and not necessarily to lay out the entire page.

A flex container is an element on a page that contains flex items. All direct child elements of the flex container are the flex items. Some properties of flexbox apply to containers while others apply to items. The primary properties for the container are flex or inline-flex. Once an item is a flex container, there are several properties to apply to the child items:

  • justify-content
  • align-items
  • flex-grow
  • flex-shrink
  • flex-basis
  • flex
  • flex-wrap
  • align-content
  • flex-direction
  • flex-flow

Flex-Wrap

Flex-wrap specifies how flex items will move to the next line when necessary. Flex-wrap accepts three values:

  1. wrap: child elements that don't fit on the row will move down to the next line
  2. wrap-reverse: same as wrap except the last item will move above while the rest of the items move to the next line
  3. nowrap: default behavior; prevents items from wrapping; items will shrink to fit the width of the parent element
